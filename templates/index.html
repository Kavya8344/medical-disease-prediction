Project Overview

This application predicts disease using patient medical information and a trained Machine Learning model.

Health Parameters

Prediction is based on Age, BMI, Blood Pressure, Glucose, Cholesterol, Heart Rate and lifestyle factors.

Technologies

Built using Python, Flask, Scikit-learn, Pandas, HTML, CSS and Bootstrap.

Dataset Features

  • Age
  • Gender
  • BMI
  • Blood Pressure
  • Glucose Level
  • Cholesterol
  • Heart Rate
  • Smoking
  • Alcohol
  • Physical Activity
  • Family History
