About the Project

Project Overview

The Medical Disease Prediction System is a Machine Learning application that predicts whether a patient is likely to have a disease based on various health parameters.

The goal is to provide quick predictions using a trained Logistic Regression model.

Technologies Used

  • Python
  • Flask
  • Scikit-learn
  • Pandas
  • NumPy
  • Bootstrap 5
  • HTML & CSS

Dataset Features

  • Age
  • Gender
  • BMI
  • Blood Pressure
  • Glucose Level
  • Cholesterol
  • Heart Rate
  • Smoking
  • Alcohol
  • Physical Activity
  • Family History

Objective

This project demonstrates how Machine Learning can assist in disease prediction using patient medical data. It focuses on data preprocessing, model training, prediction, and deployment through a Flask web application.
